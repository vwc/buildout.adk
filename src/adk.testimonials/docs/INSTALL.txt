adk.testimonials Installation
-----------------------------

To install adk.testimonials using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``adk.testimonials`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        adk.testimonials
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
